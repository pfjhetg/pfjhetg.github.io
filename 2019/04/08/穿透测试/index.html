国内外 Ping 测试

如果国内无法 Ping 国外可以 Ping，那么此 IP 一定是被封了；而如果国内外都可以 Ping，那么此时就无法判断出是否已经被封了，还需要进行下一步端口扫描测试

国内外端口扫描测试

后面需要用到机器的 IP 地址和 SSH 连接端口，IP 地址大家都知道，SSH 连接端口默认是 22 端口，如果大家没有手动更改过此端口，那么下文中需要填写 SSH 连接端口时填写 22 即可

  • 国内测试

    国内端口扫描站（http://coolaf.com/tool/port），如果端口是关闭则表明IP可能被封。

  • 国外测试

    国外端口扫描站（https://www.yougetsignal.com/tools/open-ports），如果端口是开启，而在国内测试中是关闭则可以确定IP完全被封了。而如果显示此 SSH 连接端口依旧为关闭状态，那么就检查下机器是不是在正常运行以及端口是否填写错误。
